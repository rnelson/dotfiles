About
-----
All the cool kids do it, right?

Installation
------------
### Requirements
+ Bash (`setup.sh`)
+ Git (checkout/updating)

### Install
    $ git clone git://github.com/rnelson/dotfiles.git
    $ cd dotfiles
    $ vim setup.sh
    $ ./setup.sh
    $ vim ~/.gitconfig  # you may not be me
